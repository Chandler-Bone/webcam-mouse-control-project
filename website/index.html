Cursor movement on object motion





Cursor movement on object motion is a software developed during the covid-19 crisis. The idea behind the software is to help slow down or stop the spread of the virus by limiting contact with objects. This can be extremely helpful in a hospital setting, where doctors who come in contact with covid patients can document any necessary information without actually touching a keyboard or a mouse.
